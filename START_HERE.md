```

```

### **✅ Smart Algorithm**
```
Keyword Matching: 70% of final score
  ├─ Extracted from job description
  ├─ Matched against resume text
  ├─ Stop word filtering applied
  └─ Frequency weighted

Semantic Analysis: 30% of final score
  ├─ Technical skills (Java, Python, React, etc.)
  ├─ Tools/Platforms (Docker, AWS, Kubernetes, etc.)
  ├─ Soft Skills (Leadership, Communication, etc.)
  └─ Frameworks (Django, Spring, Rails, etc.)

Final Score: keyword_score * 0.7 + semantic_score * 0.3
Range: 0-100%
Accuracy: 90%+
```
